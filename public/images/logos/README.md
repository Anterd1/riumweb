# Logos de Empresas

Esta carpeta contiene los logos de las empresas clientes que se muestran en la sección "Trusted Clients".

## Archivos requeridos:

- `BBVA.png` - Logo de BBVA
- `grupo-salinas.png` - Logo de Grupo Salinas
- `televisa-n+.png` - Logo de Televisa N+
- `expansive.png` - Logo de Expansive
- `seguros-gcp.png` - Logo de SegurosGCP
- `la-puerta-quetzalcoatl.png` - Logo de LaPuerta De Quetzalcoatl
- `harinas-elizondo.png` - Logo de Harinas Elizondo
- `fundacion-donde.png` - Logo de Fundación Dondé

## Especificaciones recomendadas:

- **Formato**: PNG con fondo transparente
- **Tamaño**: Máximo 200px de ancho
- **Altura**: Proporcional, máximo 80px de alto
- **Fondo**: Transparente preferiblemente
- **Estilo**: Versión en escala de grises (se aplicará automáticamente)

## Nota:

Si un logo no se encuentra, se mostrará el nombre de la empresa como texto de respaldo.
