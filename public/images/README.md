# Imágenes

Coloca aquí las imágenes estáticas del proyecto.

**Ejemplo:**
- `hero-bg.jpg` - Imagen de fondo para el Hero
- `hero-bg.png` - O cualquier otro formato

**Cómo usar:**
- Si colocas `hero-bg.jpg` aquí, la ruta será `/images/hero-bg.jpg`
- Vite sirve automáticamente los archivos de la carpeta `public/`
